### 介绍

使用轻量级CNN模型进一步将RFB模块组装到SSD顶部，构建RFB网络探测器。实现对变电站内电力设备（电力杆塔，输电线）的异常进行监测，如锈蚀，堆放杂
物，绝缘子破损，覆盖异物等。


模型参考文献：https://arxiv.org/pdf/1711.07767.pdf
